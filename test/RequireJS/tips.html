RECOMENDACIONES


Requerimientos mínimos para un diagnóstico óptimo:


1. Para un diagnóstico óptimo debe estar conectado a internet mediante cable de red.

2. Utilizar algún navegador como; Chrome, Safari, Firefox, Opera.

3. Android 4.0+ / macOS 10+ / GNU/Linux / Windows / Windows Phone.

4. Conocer su plan de internet.

5. Permitir geolocalización.

Equipo VelocidadDeInternet.cl


Recomendaciones para obtener mejores resultados


1. Realizar la medición mediante conexión por cable. (A excepción de equipos móviles conectados a 2/3/4G.

2. Desconectar todos los equipos de internet a excepción del equipo que realizará el diagnóstico.

3. Realizar 3 veces el diagnóstico (mañana,tarde,noche), para poder entregar información detallada.

4. No descargar información mientras se realiza el test.

5. Repetir el test varios días para obtener mejores resultados.

Equipo VelocidadDeInternet.cl


Datos del diagnóstico


Los valores obtenidos por el diagnóstico son estimaciones de nuestro sistema, por lo que puede presentar variaciones respecto a otras mediciones.

El valor asignado a cada Mbps de internet, se obtiene de la suma de los valores promedios de cada ISP (Chile). A su vez cada valor promedio se obtiene tomando el plan de internet hogar más alto y diviendo los Mbps por el costo final.

Además dentro de la sección mapas, se puede revisar la información de la red en general, para realizar distintos tipos análisis.
